# 增强型RAG（Retrieval-Augmented Generation）系统

一个结合文档检索与网络搜索的智能问答系统，能够智能判断问题类型并选择最佳回答策略。

## 系统概述

本系统是一个智能问答代理，集成了本地文档检索和实时网络搜索功能。通过智能路由机制，系统能够自动判断用户问题是否与本地文档相关，从而选择最合适的回答策略：对于文档相关问题使用RAG技术生成回答，对于非文档问题调用百度搜索API获取实时信息。

## 代码执行
python langchain_demo_rag.py

python llm_learning_demo\transformers_demo.py

python llm_learning_demo\vllm_demo.py


## 核心特性

### 🧠 智能路由机制
- **自动问题分类**：基于余弦相似度自动判断问题与文档的相关性
- **双模式回答**：文档模式（RAG） + 搜索模式（网络检索）
- **可配置阈值**：支持调整相关性判断阈值（默认0.6）

### 📚 文档处理能力
- **多格式支持**：支持网页文档加载（WebBaseLoader）
- **智能分块**：递归字符分割，优化上下文管理
- **向量化存储**：使用FAISS实现高效的相似度检索

### 🌐 实时搜索集成
- **百度搜索API**：获取最新网络信息
- **结果过滤**：支持时间范围限制（最近一周）
- **格式化输出**：结构化展示搜索结果

### 🔧 模块化设计
- **可扩展架构**：易于添加新的文档源和搜索工具
- **配置化管理**：环境变量管理API密钥
- **错误处理**：完善的异常处理机制
